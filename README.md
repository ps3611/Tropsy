## Get Started

1. Clone repository
```sh
> git clone https://github.com/ps3611/tropsy.git
```
For each directory *cron*, *server* and *client*, do:

2. Install dependencies
```sh
> npm install
```

3. Configure environment

```sh
> cp .env.example .env
# Edit .env
```

4. Launch directory's code

```sh
> npm start
```
